Documentation

Overview

The API allows users to interact with shared thoughts.
You can retrieve, create, update, and delete thoughts.
Additionally, it support image uploads.

REST

Base url: localhost:5115/Thoughts

Endpoints

GET/Thought

Retrieve all thoughts.

  • Request:

    GET localhost:5115/Thought

  • Response:

    200 OK

     
        [
            {
                "id": 1,
                "thought": "We Are Great!"
            }
        ]

GET/Thought{id}

Retrieve thought by it's ID.

  • Request:

    GET localhost:5115/Thought/{id}

  • Parameters:

    id (integer)

  • Response:

    200 OK

     
        [
            {
                "id": 1,
                "thought": "We Are Great!"
            }
        ]

GET/Trump/GetByThought/{thought}

Retrieve thoughts by specific text.

  • Request:

    GET localhost:5115/Thought//GetByThought/{thought}

  • Parameters:

    thought (string)

  • Response:

    200 OK

     
        [
            {
                "id": 1,
                "thought": "We Are Great!"
            }
        ]

POST/Thought

Create a new Trump thought.

  • Request:

    POST localhost:5115/Thought/

  • Request body: 
    {"thought": "Future"}
  • Response:

    • 200 OK

    {"id":1, "thought": "Future"}

PUT/Thought

Update an exsisting thought.

  • Request:

    PUT localhost:5115/Thought

  • Request body: 
      {"thought":"Let's have cozy weekend"}
  • Response:

    200 OK

DELETE/Thought/{id}

Delete thought.

  • Request:

    DELETE localhost:5115/Thought{id}

  • Parameters:

    id (integer)

  • Response:

    200 OK

POST/UploadingImage

Upload an image.

  • Request:

    POST localhost:5115/UploadImage

  • Parameters:

    A file (image)

  • Response:

    200 OK
